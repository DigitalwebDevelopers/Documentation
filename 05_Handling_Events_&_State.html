State is a built-in object that stores dynamic data within a component. Unlike props, which come from the outside, state is managed internally by the component.

You create state using the useState() hook: 

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return <h2>Count: {count}</h2>;
}

count is the current state value. setCount is the function used to update it. Every time you call the update function, React re-renders the component.

  const [count, setCount] = useState(0);
