Understanding useReducer Hook

useReducer() is a React hook that helps manage state using a reducer function. Instead of directly updating values, you send an ‚Äúaction‚Äù and the reducer decides how to update the state.

üî∏ Basic Syntax of useReducer Hook

The state holds the current data. The dispatch function sends actions to update it. 

const [state, dispatch] = useReducer(reducer,initialState);

These actions go through a reducer ‚Äî a function that returns a new state based on the current state and action. The initialState sets the starting value.

üî∏ Simple Counter Example

This code defines a simple counter using the useReducer() hook. It starts with an initial state of 0. The reducer function handles two actions: increment and decrement

which update the count accordingly. The component displays the count and two buttons to increase or decrease it using the dispatch function. 

import React, { useReducer } from 'react';

const initialState = 0;

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return state + 1;
    case 'decrement':
      return state - 1;
    default:
      return state;
  }
}

function Counter() {
  const [count, dispatch] = useReducer(reducer, initialState);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => dispatch({ type: 'increment' })}>Increase</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>Decrease</button>
    </div>
  );
}

üî∏ Real Use Case: Form State

This code demonstrates a form using the useReducer() hook to manage its state. The initial state is an object with two properties: name and email.

The formReducer function updates the respective field when the user types into the input fields. The handleChange function dispatches an action to update the state with the new value for the corresponding input field.

The form inputs are controlled, with their values coming from the formData state. 

const initialForm = { name: '', email: '' };

function formReducer(state, action) {
  return {
    ...state,
    [action.field]: action.value
  };
}

function MyForm() {
  const [formData, dispatch] = useReducer(formReducer, initialForm);

  const handleChange = (e) => {
    dispatch({ field: e.target.name, value: e.target.value });
  };

  return (
    <form>
      <input name="name" value={formData.name} onChange={handleChange} />
      <input name="email" value={formData.email} onChange={handleChange} />
    </form>
  );
}

üî∏ Why we use useReducer Hook

You have multiple state values that change together. The state update logic is complex. You want to make your logic more readable and testable
