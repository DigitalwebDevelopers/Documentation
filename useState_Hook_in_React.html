Understanding useState Hook

React is a popular JavaScript library for building user interfaces. In modern React (using functional components), one of the most essential tools for managing component state is the useState() Hook.

What is useState Hook in React ?

useState() is a Hook provided by React that allows you to add state to your functional components. Traditionally, state was only available in class components. But with React Hooks (introduced in React 16.8), you can now use state inside functional components. 

const [state, setState] = useState(initialValue);

Breakdown of the Given Code :

The useState() hook takes an initial value and gives back the current state and a function to change it.

You can name them anything, but names like count and setCount keep the code easy to understand. Updating the state re-renders the component.

ðŸ”¸ Basic Example : An Counter App

useState(0) sets the starting count to 0. When setCount(count + 1) is called, it increases the count by 1 and React updates the component automatically. 

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0); // count starts at 0

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>Increase</button>
    </div>
  );
}

You can call useState() multiple times in a single component to manage different pieces of state: 

function Profile() {
  const [name, setName] = useState('John');
  const [age, setAge] = useState(30);

  return (
    <div>
      <p>Name: {name}</p>
      <p>Age: {age}</p>
      <button onClick={() => setAge(age + 1)}>Increase Age</button>
    </div>
  );
}

ðŸ”¸ Working with Objects in State

When using objects as state, always update them immutably using the spread operator. If you just did setUser({ name: 'Bob' }), you would lose the age property. 

const [user, setUser] = useState({ name: 'Alice', age: 25 });

function updateName() {
  setUser({ ...user, name: 'Bob' }); // keeps age unchanged
}
