ðŸ”¸ Handling Forms in React

React forms work by storing input values in state and updating them with events.

The inputâ€™s value is linked to name state. As the user types, handleChange() updates the state. This is called a controlled component. 

import React, { useState } from 'react';

function MyForm() {
  const [name, setName] = useState('');

  const handleChange = (e) => {
    setName(e.target.value);
  };

  return (
    <form>
      <input type="text" value={name} onChange={handleChange} />
      <p>You typed: {name}</p>
    </form>
  );
}
