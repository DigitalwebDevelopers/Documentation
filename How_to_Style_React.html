1. Regular CSS Files

The simplest way to style is to create a .css file and import it into your component: 

/* styles.css */
.title {
  color: blue;
  font-size: 24px;
}

import './styles.css';

function App() {
  return <h1 className="title">Hello World</h1>;
}

You can keep styling separate and use traditional CSS features like media queries and animations.

2. Inline Styles

You can apply styles directly using the style attribute. Note that the value is an object, and CSS properties use camelCase: 

const style = {
  color: 'green',
  fontWeight: 'bold'
};

<h2 style={style}>Inline Styled Text</h2>

Useful for dynamic styling but can become hard to manage in large components.

3. CSS Modules

CSS Modules scope styles locally by default, which avoids class name conflicts: 

/* App.module.css */
.button {
  background-color: orange;
}

import styles from './App.module.css';

<button className={styles.button}>Click Me</button>

This is ideal for medium to large projects.

4. Styled Components

A powerful method using tagged template literals to style components: 

npm install styled-components

import styled from 'styled-components';

const Button = styled.button`
  background: purple;
  color: white;
`;

<Button>Styled Button</Button>

Styled-components let you co-locate styles and logic in one file and support dynamic styling using props.
