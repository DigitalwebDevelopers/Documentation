Understanding useContext Hook

The useContext() hook is a way to read shared data (called "context") without manually passing props down the component tree. It works with the Context API to let any component access global values â€” like theme, user info, language settings, etc.

How The useEffect Hook Works ?

1. Create the Context

You start by using createContext() to make a context object. 

import { createContext } from 'react';
const ThemeContext = createContext();

2. Wrap Your Components with the Provider

The Provider lets you pass a value to the context. All components inside this provider can now access that value. 

<ThemeContext.Provider value="dark">
  <MyComponent />
</ThemeContext.Provider>

3. Access Context in a Child Component

Use useContext() to consume the value inside any nested component. 

import { useContext } from 'react';
const theme = useContext(ThemeContext);
